          헤더

          헤더 팔로우

          사이드바와 주제 탭을 함께 숨겼을 때 헤더에 한 번에 표시할 팔로우 라이브 채널 수를 정합니다.

          헤더 바로가기 (사이드바 숨김 시)

          사이드바를 숨기면 선택한 항목을 헤더에 아이콘으로 표시해 바로 이동할 수 있습니다. 체크하면 표시됩니다.

          사이드바

          사이드바 항목별 숨김

          '사이드바 숨김'이 켜져 있으면 사이드바 전체가 가려져 항목별 설정은 적용되지 않습니다.

          팔로우 채널 자동 갱신

          사이드바가 펼쳐진 상태에서 팔로우 채널 목록을 주기적으로 새로고침해 라이브 상태를 갱신합니다(접힘 상태에선 동작하지 않습니다). 커스텀은 3~600초이며, 10초 이하로 설정하면 호출 제한을 피해 설정값과 설정값~10초 랜덤을 번갈아 호출합니다.

          채팅창 정리

          • %

          채팅창이 열려 있는 라이브/다시보기에서 동작합니다. 너비는 채팅창 왼쪽 경계를 드래그해 조절하며 전역으로 저장됩니다.

          배지 모아 챗 확장이 채팅창을 제어 중이면 충돌을 막기 위해 이 기능들이 자동으로 양보합니다(배지 모아 챗이 켜진 동안 우리 채팅 정리는 적용되지 않음). 배지 모아 챗을 끄면 다시 적용됩니다.

          통나무파워

          • ‘적립 중’ 표시 위치 적립되는 동안 ‘적립 중’을 어디에 보일지: 끔 / 배지(인라인) / 호버 툴팁
          • 1시간 타이머 표시 위치 1시간 보상까지 남은 시간을 어디에 보일지: 끔 / 배지(인라인) / 호버 툴팁

          다른 확장이 채팅창에 이미 통나무파워를 표시 중이면 중복을 막기 위해 이 통나무파워 표시가 자동으로 양보합니다(상대 확장이 켜진 동안 우리 배지는 나타나지 않음). 상대 확장을 끄면 다시 표시됩니다.

          배지 클릭 동작

          • 개

          라이브 페이지에서 동작합니다. 보유 표시는 채팅 입력창 옆에 현재 채널의 보유량을 보여주고, 자동 획득은 시청 보상(1시간 시청 등)을 자동으로 받습니다. 획득 토스트는 1시간 보상을 받았을 때 알림을 띄웁니다. 배지를 클릭하면 위에서 고른 동작을 실행합니다.
